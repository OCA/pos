This module is designed to display product warnings while processing payments in the POS system.

**Table of contents**

.. contents::
   :local:

Usage
=====

Open the "Sales" tab in the Product form and navigate to the Point of Sale section. Add warning text to the POS warning message field.

Whenever this product is selected in POS, a warning will be shown during payment.
