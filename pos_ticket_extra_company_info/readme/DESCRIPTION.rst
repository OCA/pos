This module extends the functionality of Point of Sale module.

It adds missing company fields on PoS ticket.

**In Proxy Mode, it adds:**

- company address
- company registry

**In Non Proxy Mode, it adds:**
- company address
- company VAT ID
- company Registry
- company website
- company email
