Configuration
=============

To configure this module, you need to:

* Go to 'Point Of Sale' / 'Configuration' / 'Payment Methods'

* Select Payment Method, edit and set an image

.. figure:: https://raw.githubusercontent.com/OCA/pos/13.0/pos_payment_method_image/static/description/payment_method_form.png

Known issues / Roadmap
======================

This module changes the pos_journal_image (12.0) to pos_payment_method_image (13.0) because of the model changing.

Migration script from 12.0 is not covered yet.
